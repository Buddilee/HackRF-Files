## 🚀 What Can You Do with This?
If you own a **HackRF Portapack**, you can upload these files to your device, unlocking a variety of remote controls for:
- **Lighting Systems** 💡
- **Doorbells** 🚪
- **Cars** 🚗
- **And much more!**

-----

## 📦 How to Install?

- **Move** all files from the repository to the **root directory** of your **HackRF's SD card**, ensuring that any existing files are replaced.
